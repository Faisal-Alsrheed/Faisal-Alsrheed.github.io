# Faisal Alsrheed - Personal Website

This repository contains the source code for my personal portfolio website, hosted via GitHub Pages at [faisal-alsrheed.github.io](https://faisal-alsrheed.github.io).

## Overview

The site is a simple, static HTML/CSS single-page website that highlights my:
- Executive Summary
- Professional Experience
- Open Source Contributions (Keras, Google Generative AI)
- Kaggle Competition rankings
- Talks and Podcasts
- Educational Background
- Contact Links (Kaggle, LinkedIn, GitHub)

## Technologies

- HTML5
- CSS3

## Deployment

Changes pushed to the `main` branch will automatically be deployed to GitHub Pages.
